### Built With

- [React.js](https://reactjs.org/)

- [Styled-components](https://styled-components.com/)
- [Chakra UI](https://chakra-ui.com/)

You can run this project by.

1.  Clone the repo

```sh
git clone https://github.com/Sabian329/CosyMovies
```

2.  Install NPM packages

```sh
npm install
```

3.  Run the localhost

```sh
npm start
```
